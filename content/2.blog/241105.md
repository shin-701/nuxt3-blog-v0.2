## シンタックスハイライトも！

シンタックスハイライト用ライブラリの[Shiki](https://shiki.style/)が組み込まれているので、テーマを設定してハイライトをつけられます。

```js [aaaa.js]
const message: string = 'Hello, World!';
```

```vue
<template>
  <Hero />
  <ContentRenderer
    v-if="data"
    :value="data"
  />
  <div v-else>
    <h1>Page not found</h1>
  </div>

    <h1>これは見出しです</h1>
    <p>これは段落です。適切な余白・行間があるか確認してください。</p>

    </ul>
```
